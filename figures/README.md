# Figures (generated)

These figures are produced from the source feeds by a single command:

    python scripts/run_all.py

Each exhibit script writes both a `.pdf` (embedded by the manuscript) and a `.png` (for quick
viewing) into this folder. The contents are gitignored so the git repository tracks source only;
the **released archive bundles reference copies** so the results can be inspected, and the
manuscript compiled, without re-running the pipeline.

    fig1_regime          recovery as a regime change (systemwide index vs 2022, subway anchor)
    fig2_dow_profile     day-of-week shape, pre-shock vs recent, and current shape by mode
    fig6_daytype_ratios  Saturday- and Sunday/Holiday-to-weekday ratios over time, by mode
    fig4_recovery_ratio  recovery vs the 2022 and 2019 baselines, by mode
    fig5_decomposition   subway trend collapse/plateau and seasonal-amplitude compression
    fig6_modemix         rider-mode share evolution (2021 onward)
    fig7_supply_demand   SIR supply vs demand and the service-per-rider ratio
